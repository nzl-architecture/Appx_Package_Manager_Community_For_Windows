# Appx Package Installer Community
</div>

###  说明
一个安装appx、msix等打包格式的小工具, 用于在右键菜单添加快捷安装选项，可选择系统级安装或用户级安装，还可以通过注册xml来把松散文件注册成microsoft store（容器）应用.

注意:

* xml注册需要在系统设置打开开发者选项才能使用.

### 使用方法
从release 下载main.zip文件（从github actions构建得到），完全解压后运行install.bat即可安装
